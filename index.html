关于踝关节骨折

踝关节骨折是急诊中常见的损伤，约占所有骨折的9-18%。踝关节由胫骨、腓骨远端和距骨组成，其稳定性依赖于骨骼结构和周围韧带组织。

本指南涵盖踝关节骨折的病理生理学、流行病学、诊断方法、治疗选择及完整的康复方案，为医疗专业人员提供全面参考。

一、踝关节骨折

1. 病理生理学

  • 历史背景: 1856年Dr. Sir Percival Pott从马背跌落导致踝关节开放性损伤，骨组织外露
  • 治疗创新: 首次采用保守治疗（非截肢），通过固定患肢位置实现自愈
  • 命名由来: 这种特殊踝关节骨折因此被称为"POTT骨折"
  • 临床意义: 该案例改变了传统创伤处理方式，为现代康复医学奠定基础
  • 技术发展: 近30年才广泛应用于骨折治疗
  • 治疗变革: 逐渐取代单纯石膏固定的传统方法
  • 技术特点: 通过手术实现解剖复位和机械稳定

核心原则: 康复方案需同时评估骨折和软组织损伤程度

内侧结构:

  • 血管风险: 胫后动脉损伤可能
  • 神经风险: 胫神经易受损
  • 韧带损伤: 三角韧带（维持足弓结构的关键）

外侧结构:

  • 肌腱损伤: 腓骨长/短肌腱
  • 韧带损伤: 外侧副韧带
  • 神经损伤: 腓浅神经和腓肠神经

损伤评估:

  • 单纯腓骨骨折影响较小（非承重骨）
  • 胫腓骨联合骨折需特殊处理
  • 骨折位置（近端/远端）决定康复方案

功能影响:

内侧踝关节骨折需评估胫后动脉损伤风险和胫神经损伤可能，三角韧带损伤将直接影响足弓稳定性。

外侧骨折需关注腓骨肌腱（腓骨长肌/短肌）状态、外侧副韧带完整性以及腓浅神经/腓肠神经损伤情况。

Weber分类系统:

Type A:

  • 横形骨折
  • 三角韧带可能受牵拉但完整
  • 内踝骨折通常稳定

Type B:

  • 螺旋形/旋转型骨折
  • 常伴内踝骨折和三角韧带损伤
  • 多需内固定治疗

Type C:

  • 高位踝关节损伤
  • 骨折线向近端延伸
  • 必须进行ORIF手术

康复变量:

  • 骨折类型决定负重时间
  • 软组织损伤程度影响活动范围训练
  • 神经血管状态决定康复进度

临床决策:

  • 单纯腓骨骨折可早期活动
  • 合并胫骨骨折需延长保护期
  • 高位骨折(C型)需严格遵循手术医生方案

2. 流行病学

  • 发生率: 急诊室中9-18%的骨折为踝关节骨折
  • 常见类型: 外翻骨折最常见，单踝骨折占52%，双踝骨折占36.6%，三踝骨折占11.4%
  • 易感人群: 年轻男性运动员、中年女性/孕妇
  • 年龄相关机制: 年轻人多导致韧带撕裂，年长者多导致骨折

3. 诊断

典型表现

  • 外伤史: 绝大多数有明显急性创伤事件
  • 负重痛: 患肢承重时疼痛显著
  • 肿胀: 损伤部位可见明显肿胀

影像学检查

  • X线检查: 至少需要三个投照体位（正位、侧位、踝穴位）
  • MRI指征: 当怀疑合并韧带损伤（如三角韧带）、肌腱损伤或软骨损伤时必需

4. 非手术治疗

治疗选择

  • • 2年随访时保守治疗与ORIF手术的临床结局相似
  • • 美国倾向于优先尝试非手术治疗

初期固定

  • • 单纯腓骨骨折: 推荐使用可拆卸支具
  • • 胫骨骨折: 需严格非负重石膏固定3-4周

康复进程

  • • 8-12周内渐进性增加负重（WBAT原则）
  • • 术后/保守治疗后的康复训练方案基本相同

特殊考量

腓骨骨折愈合快，应尽早开始部分负重训练。在能提供足够保护的前提下，优先选择限制更小的固定方式。

5. 手术指征

  • 不稳定/移位骨折：特别是伴有距骨半脱位的情况，当距骨与胫骨对位不良时，必须进行手术治疗
  • 骨折分型：Weber分类中的B型和C型骨折，或II-III阶段骨折
  • 旋前型骨折：这类骨折患者在负重时会出现扁平足表现，通常涉及三角韧带损伤
  • 紧急复位需求：需要尽快复位并稳定的骨折病例
  • 临床依据：Kandelwal和Regier等学者2015年的研究支持上述手术指征

6. 手术技术

  • 器材选择依据：完全取决于骨折的具体部位和严重程度
  • 常用器材：包括但不限于钢针、螺钉、钢板、髓内钉和固定棒等
  • 组合使用：临床中常根据实际情况组合使用多种固定器材
  • 技术要点：通过腓骨外侧向胫骨打入固定物
  • 最佳固定方案：使用2枚4.5mm螺钉可提供最强的固定效果
  • 解剖关系：该技术实质是将腓骨与胫骨进行固定融合
  • 临床确认：术后X线检查首要确认是否实施了联合固定
  • 活动度限制：联合固定会限制胫腓骨之间的正常分离活动
  • 背屈机制：正常背屈时胫腓骨需产生约20∘的分离间隙
  • 临床影响：固定后患者可能无法达到健侧相同的背屈角度
  • 康复考量：这对步态和上下楼梯功能有显著影响
  • 评估重点：需明确固定物是否连接了胫骨和腓骨
  • 活动机制：距骨在胫骨上滑动时，需要胫腓关节的协调活动
  • 硬件影响：若固定物限制胫腓分离，将直接影响背屈活动度
  • 康复预判：术前了解固定方式对制定康复计划至关重要
  • 典型表现：无胫腓连接的患者背屈活动度通常优于有连接者

7. 手术并发症

感染风险因素：

  • 总体发生率1.44%，但特定人群风险显著升高
  • 开放性损伤患者感染风险最大
  • 糖尿病患者感染率达7.71%
  • 周围血管疾病患者
  • 吸烟者（尼古丁影响组织愈合）

其他并发症：

  • 内固定物移位（美国约15%需取出）
  • 钢板末端二次骨折
  • 踝关节生物力学改变（联合固定患者更明显）
  • 背屈活动度永久性降低

8. 保护运动期

核心任务：

  • 疼痛管理：重点监测肿胀程度（前4周关键指标）
  • 创口保护：密切观察手术部位，预防感染
  • 心肺维持：针对老年患者设计低强度有氧运动

特殊考量：老年患者需特别关注心血管系统适应性训练

负重原则：

  • 标准方案：严格非负重4周
  • 改良方案：2周后可开始部分负重（需医生评估）
  • 使用CAM靴时保持踝中立位，全天穿戴

东西方差异：

  • 单纯腓骨骨折：西方可能早期负重（腓骨非承重骨负重不会影响骨折愈合或内置物稳定性）
  • 中国临床更倾向保守方案

生物力学原理：踝关节运动时腓骨会上下移动（内翻上移，外翻下移）

训练内容：

  • 移动训练：拐杖/助行器使用教学（含楼梯技巧）
  • 关节活动：
    • 髋膝关节全范围活动训练
    • 脚趾主动/被动活动（毛巾抓握、弹珠拾取）
  • 肌力训练：重点加强膝关节周围肌群

特殊干预：

  • 可拆卸靴患者：进行踝关节被动活动
  • 疤痕处理：早期开始疤痕松解术
  • 有氧选择：下肢自行车（需评估负重状态）

9. 渐进性承力/力量训练

  • 时间范围: 第5-10周开始增加负重训练
  • 核心目标:
    • 增加踝关节跖屈/背屈活动范围（Plantarflexion/Dorsiflexion ROM）
    • 监控指标：需持续观察疼痛、肿胀情况，特别注意疤痕状态
  • 功能进展: 此阶段末期应达到完全负重状态（需医生确认）

肿胀管理:

  • 采用冰敷和抬高下肢（因足踝离心脏最远，肿胀可能持续1年+，老年人更明显）
  • 靴具移除后使用压力袜/套（需保证适度压力，避免限制肌肉收缩功能）

步行训练:

  • 逐步增加负重状态下的步态训练
  • 重点改善背屈活动度（Dorsiflexion）和膝关节伸展能力

疼痛控制: 注意靴具可能因肿胀导致不适，需调整疼痛管理方案

核心干预:

  • 进阶承重步行训练→短腿步行石膏支具过渡
  • 踝关节被动/主动活动训练（PROM/AROM）

全身方案:

  • 实施心血管训练计划（如自行车）
  • 开展全身力量训练（侧重下肢但不限于）

治疗重点: 60%时间用于步态训练，同步监测关节活动度进展

10. 功能/运动感觉训练

  • 时间节点: 伤后10-14周（约2个月后）
  • 关键指标:
    • 实现踝关节全范围活动度（需达到主动ROM完全恢复）
    • 各种平面步态正常化（4个月时应基本恢复）
    • 通过自行车等有氧训练维持心血管健康
  • 特殊注意: 持续控制水肿（压力袜/支具辅助）

关节松动重点:

  • 距骨关节（Talocrural）背屈松动（优先）
  • 检查距下关节旋前/旋后功能（因长期制动易僵硬）
  • 胫腓关节活动度评估（无贯穿螺钉者可实施）

强化训练:

  • 非负重肌力训练（如弹力带）
  • 器械训练（腿举机渐进负荷）

平衡基础:

  • 从Romberg测试起步
  • 逐步过渡到串联站立（Tandem stance）
  • 避免早期单腿站立（考虑跌倒风险）

11. 运动专项训练

进阶内容:

  • 多平面平衡/本体感觉训练（不平整表面）
  • 下台阶离心训练（重点干预项目）
  • 不同地面跑步训练

训练频率:

  • 力量训练3-4次/周（持续至4-5个月）
  • 老年患者需加强平衡训练（针对跌倒史）

出院标准:

  • 年轻患者可侧重专项技能
  • 老年患者需确保家庭训练计划（建议门诊随访）

二、知识小结

知识点 核心内容 易混淆点/关键细节 临床相关性
踝关节骨折分类 Weber分类系统：
- Type A：横向韧带损伤，韧带完整
- Type B：螺旋骨折，累及内侧踝和三角韧带
- Type C：高位踝损伤，需手术固定		 Type A与B的损伤机制差异（旋转 vs 横向力） 52%累及腓骨，36.6%双踝骨折，11%三踝骨折
Pott骨折 历史上首个保守治疗的开放性踝骨折案例（1856年），源自骑马坠落，骨突出后自行复位 术语混淆：Pott骨折特指严重踝骨折，但现代分类更细化 现代多需手术，保守治疗仅适用于稳定型
手术指征 不稳定/移位骨折、距骨半脱位、涉及三角韧带的旋前型骨折 下胫腓联合固定术（螺钉连接胫腓骨）影响背屈活动度 术后15%需取出内固定物（疼痛或活动受限）
康复阶段 Phase 1（0-4周）：控制肿胀、非负重活动、心血管训练
Phase 2（5-10周）：渐进负重、关节活动度训练
Phase 3（>2月）：全负重、力量与平衡训练		 腓骨骨折可早期负重（非承重骨），但需保持中立位 老年患者需长期关注平衡（防跌倒）
并发症 感染率1.44%（吸烟/糖尿病者风险高）；内固定物移位；踝关节生物力学改变（尤其下胫腓联合固定后） 三角韧带损伤易被忽视，需MRI确认 术后肿胀可持续1年以上（正常现象）
治疗对比 手术 vs 保守：2年预后相似，但手术适用于不稳定骨折；中国 vs 美国：中国更倾向手术，美国部分腓骨骨折用行走靴 保守治疗需3-4周非负重，8-12周渐进负重 老年患者康复重点：平衡训练（防二次跌倒）
